# Smart_Recipe_Finder
1 Overview
When you don’t understand what to cook using the ingredients you have, you can use this website. 
You just select the ingredients you have, and it will give you recipe suggestions according to them.

2 features
• To minimize food wastage by suggesting recipes using available ingredients. 
• To provide personalized meal recommendations based on user preferences. 
• To allow users to explore a wide range of cuisines and dietary options. 
• To enhance user experience through AI-powered search and filtering options.

3 Technologies used
• Frontend: HTML, CSS, JavaScript 
• Backend:  Python (Flask) 
• Database: SQLite 
• APIs: Spoonacular API for recipe data, Google Vision API for image recognition
